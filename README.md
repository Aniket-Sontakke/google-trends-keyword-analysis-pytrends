# Google Trends Keyword Analysis using Pytrends and Python

## Project Overview
This project analyzes keyword search trends using Google Trends data through the Pytrends library.  
The goal is to explore how keywords perform across countries, time periods, and related searches using data visualization techniques.

## Problem Statements Covered
1. Build a reusable script where changing the keyword allows new trend analysis.
2. Identify the top 15 countries where the keyword is searched the most and visualize the results.
3. Plot a world map showing geographic keyword interest.
4. Analyze time-wise keyword trends across different years.
5. Compare related keywords and visualize their performance.

## Tools and Technologies
- Python
- Pytrends
- Pandas
- Matplotlib
- Seaborn
- Plotly (if used for maps)

## Analysis Performed
- Keyword interest over time
- Country-wise search distribution
- World map visualization of keyword trends
- Related keyword comparison
- Multi-keyword trend visualization

## Visualizations
- Line Charts
- Bar Charts
- Geographic Maps
- Trend Comparison Graphs

## Workflow
1. Connect to Google Trends using Pytrends
2. Extract keyword trend data
3. Perform geographic and time-based analysis
4. Compare related keywords
5. Create visualizations using Python libraries
